MiniShift Tips

hard cleanup minishift environment

For example, for errors such as

domain 'minishift' already exists with uuid

  1. Check if the VM is still running

    $ sudo virsh list --all

  2. If it is running, stop the minishift VM

    $ sudo virsh destroy minishift

  3. Delete the VM

    $ sudo virsh undefine minishift

  4. Delete the .minishift/machines dir

    $ rm -rf ~/.minishift/machines.

  5. Or as an extreme step, delete all your minishift settings and start from scratch

    $rm -rf ~/.minishift
